## Advanced Token Summoner

While the base summoners default the name of the loot token to `{Share Token Name}`-loot, the Advanced Token Summoner allows the user to input a custom loot token name and symbol. This is the summoner used in the DAOhaus [Summon](https://summon.daohaus.club/) app.

[Github](https://github.com/HausDAO/Baal/blob/feat/baalZodiac/contracts/higherOrderFactories/BaalAdvTokenSummoner.sol)

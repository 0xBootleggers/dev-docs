# Shaman Examples

## ETH Onboarder Shaman

This shaman mints DAO shares or loot to anyone who sends ETH into it. This can be used as a fundraising tool that onboards DAO members while growing the treasury.

- [Contract](https://github.com/HausDAO/baal-shamans/blob/main/contracts/onboarder/EthOnboarder.sol)
- Required permission level: Manager

Other examples of [onboarder](https://github.com/HausDAO/baal-shamans/tree/main/contracts/onboarder) shamans.

## Subscription Shaman

This shaman enables a subscription model. It grants shares upon each payment, enforces payment periods, and collects late payments.

- [Contract](https://github.com/HausDAO/baal-shamans/blob/main/contracts/subscriptions/Subscriptions.sol)
- Required permission level: Manager

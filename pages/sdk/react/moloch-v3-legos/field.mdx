# Field Legos

Field legos are used to define a field used within a form lego.

## [Github](https://github.com/HausDAO/monorepo/blob/develop/libs/moloch-v3-legos/src/fields.ts)

## Usage

### Examples

**How to make a field lego**

```tsx
import { MolochFieldLego } from '@daohaus/moloch-v3-fields';

const customField = {
  id: 'customId',
  type: 'input',
  label: 'I am custom',
  placeholder: 'just holding a place',
  expectType: 'url',
};
```

Field types can be added when creating custom [`moloch-v3-fields`](/sdk/react/moloch-v3-fields). Here are the existing [fields](https://github.com/HausDAO/monorepo/blob/develop/libs/moloch-v3-legos/src/fields.ts).

`expectType` automatically validates input values. There are a few [`ValidateField`](https://github.com/HausDAO/monorepo/blob/develop/libs/utils/src/utils/validation.ts#L21) options for this available out of the box:

- `number`
- `boolean`
- `array`
- `ethAddress`
- `url`
- `email`
- `percent`
- `object`

# @daohaus/data-fetch-utils

Data fetch utilities provide a subset of utilities primarily utilized within our data libraries. These are key functions required for making API calls to fetch data from a range of sources such as subgraphs, IPFS, or any other data points we access. This library is not usually used on it's own, but in the context of [@daohaus/moloch-v3-data](/sdk/data/moloch-v3-data) and [@daohaus/profile-data](/sdk/data/profile-data).

## Usage

### Installation

```bash
yarn add @daohaus/data-fetch-utils
```

### Examples

**How to post to IPFS with Pinata**

```tsx
import { pinataPostJSON } from '@daohaus/keychain-utils';

const res = await pinataPostJSON({
  creds: {
    pinata_api_key,
    pinata_api_secret,
  },
  jsonString: JSON.stringify({ this: 'is fun' }),
});
```

# @daohaus/abi

ABI provides quick access to the main Moloch v3 contract ABIs.

## [Github](https://github.com/HausDAO/monorepo/tree/develop/libs/abi)

- [ABI List](https://github.com/HausDAO/monorepo/blob/develop/libs/abis/src/lib/abis.ts#L16)
- [ABI Files](https://github.com/HausDAO/monorepo/tree/develop/libs/abis/src/abi)

## [NPM](https://www.npmjs.com/package/@daohaus/abis)

## Usage

### Installation

```bash
yarn add @daohaus/abis
```

### Examples

**How to get ABI instances**

```tsx
import { LOCAL_ABI } from '@daohaus/abis';

const summonerContractAbi = LOCAL_ABI.BAAL_SUMMONER;

console.log(summonerContractAbi);
```

# React Features Packages

![](https://hackmd.io/_uploads/rk4FOqmRh.png)

These features are a powerful blend of UI and data elements, crafted with the intention to provide you with robust, ready-to-use components that seamlessly integrate with your applications.

We understand the complexity of building UIs that interact with blockchain data, and our aim is to simplify this process. Whether you are developing a new DAO application or enhancing an existing one, our React Features will help you accelerate your development workflow.

![](https://hackmd.io/_uploads/Sks8b3X02.png)

![](https://hackmd.io/_uploads/Sk4P-n702.png)

- **Connect**: Seamlessly establish connection with a web3 wallet.

![](https://hackmd.io/_uploads/Hk8Q7hQRh.png)

- **Moloch-v3-Hooks**: This set of hooks are used to fetch all data related to your Moloch V3 DAOs.

![](https://hackmd.io/_uploads/ryURmnmA2.png)

- **Moloch-v3-Macro-UI**: This feature provides high-level UI components designed specifically for Moloch V3 DAOs. It gives you the power to craft exceptional user experiences without having to build everything from scratch.

![](https://hackmd.io/_uploads/S164rhQ0h.png)

- **TX-Builder**, **Form Builder**, **Moloch-v3-Fields** and **Moloch-v3-Legos**: These packages allows you to easily create forms that interact with your smart contracts., create transactions and manage the transaction lifecycle for your UI.

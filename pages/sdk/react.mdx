# React Features Packages

![react-packages](https://github.com/HausDAO/monorepo/assets/522182/87703936-2034-4bee-8084-a15030b2085c)

These packages are a powerful blend of UI and data elements, crafted with the intention to provide you with robust, ready-to-use components that seamlessly integrate with your applications.

We understand the complexity of building UIs that interact with blockchain data, and our aim is to simplify this process. Whether you are developing a new DAO application or enhancing an existing one, our React Features help accelerate your development workflow.

## [connect](/sdk/react/connect)

Seamlessly establish connection with a Web3 wallet.

## [moloch-v3-hooks](/sdk/react/moloch-v3-hooks)

# DAO App Starter

Quickly start your DAO app with our starter repo and use efficient admin app functionality to unlock endless customization options with DAOHaus tooling.

The starter is written in React w/ Typescript and uses [Vite](https://vitejs.dev/) for environment tooling.

The features include:

- Single and multi-DAO capabilities
- Wallet connection with [@daohaus/connect](https://hackmd.io/@bootleggers/Skfd50_w3/https%3A%2F%2Fhackmd.io%2F%40bootleggers%2FryP60JYvn)
- DAO overview and settings pages
- Proposal list and proposal details pages
- Member list and member details pages
- All current DAOhaus admin proposal forms
- An example custom field, form, and transaction

# Subgraph Maintenance SOP

This guide will explain how to (1) deploy subgraphs to each network the DAOhaus protocol supports AND (2) how to manage the studio subgraph billing account.

This is not a guide for developing subgraphs and/or how to use them within the DAOhaus protocol.

## Resources

- [The Graph development docs](https://thegraph.com/docs/en/quick-start/)
- [Monorepo subgraph code](https://github.com/HausDAO/monorepo/tree/develop/apps/moloch-v3-subgraph)
- [Subgraph API Endpoints and explorer urls](https://docs.daohaus.club/subgraphs)

All subgraphs are deployed to The Graph's decentralized network on Arbitrum and owned by this [safe on Arbitrum](https://app.safe.global/home?safe=arb1:0x08678A8d65D8d81bD13db617211F1E9CE176B2fB)
